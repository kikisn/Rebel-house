# Rebel House Fitness Website

A modern, responsive website for Rebel House - a fitness studio in Salt Lake City, UT offering indoor cycling, HIIT, and yoga classes.

## Pages

- **Home** (`index.html`) - Main landing page with hero section, services, testimonials, and contact form
- **About** (`about.html`) - Information about the studio and team
- **Book & Buy** (`book-buy.html`) - Class booking and membership purchase options
- **Contact** (`contact.html`) - Contact information and inquiry form
- **Locations** (`locations.html`) - Studio locations and maps
- **Dashboard** (`dashboard.html`) - User dashboard for members

## GitHub Pages Deployment

This site is ready to be deployed on GitHub Pages:

1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Select the branch (usually `main`) and root directory
4. Your site will be available at `https://[username].github.io/[repository-name]/`
